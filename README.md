# League of Legends Team Builder


Una webapp sencilla de **League of Legends** que facilita la visualización de las habilidades de los campeones.

Disponible con [*GitHub Pages*](https://zergiopc.github.io/LoL-Team-Champs-view/).

## Features

- **Buscador de Champs**: Cada equipo tiene un buscador propio.
- **Ver sus Habilidades**: Ver nombres de sus habilidades de forma rapida, y a que tecla pertenecen.
- **Diseño Responsive**: Aunque no está pensado para mobiles, se creó un diseño responsive que permita su uso, pero se recomienda el uso en PC.


## How it Works

Hace uso de la API pública de _*Riot Games*_ [Data Dragon](https://developer.riotgames.com/docs/lol#data-dragon).

## How to Run

1.  Clonar el repositorio.
2.  Instalar dependencias:
    ```bash
    npm install
    ```
3.  Iniciar dev server:
    ```bash
    npm run dev
    ```
